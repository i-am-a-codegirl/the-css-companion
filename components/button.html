Button

Primary Button

Buttons are super important part of any website. You can use the button for call-to-actions and in many more ways to interact with your audience.

You just have to put class "btn" with class of sizes like xs, sm, large to get the desired size "btn large". You can also give additional functionalities to your button loike disable or hover by using classes for e.g. "btn xs disable/hover". Besides, its super easy to customize buttons as per the theme of your website.


Icon Button

Icon in Buttons are useful in displaying important functions that user might want to do. For using the icons you just need to add class btnn with class fa fa-hone(for home icon) in thr icons within the buttons. For e.g. "btnn" for buttons & on icons with class fa & fa-home.


Link Button

Youtube Webucator Google

Links buttons are created using anchor tags. These buttons are usually utilized to link to a target landing page. For e.g. you can use class "link-btn"with your anchor tags. You can change the sizes by applying the size class, simple & easy.


Float Button

Floating buttons are used to provide some special power/functionalities to the user like edit page in google docs. You can use the btn-fab class with desired icon class to create floating buttons
