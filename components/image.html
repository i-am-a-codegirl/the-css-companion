Responsive Image

Want your images to adapt to the size of the screen and beuatifully fits into any screen size. Here is responsive image component on your rescue. You just need to use class = "image-responsive" with your image tag to beautiful responsive images.

Round Image

Round images are so common in web designs, here is a round image component which you can use to create perfect round images. Use class = "image-round image-responsive"to make round images that are round and responsive.
